# Chrome Web Store Privacy Form

[privacy]
single_purpose:
YouTube Mix Blocker removes YouTube Mix recommendations from YouTube pages and cleans Mix watch URLs so videos open as regular video links. It does not replace the feed, add recommendations, change search, or modify unrelated websites.

permission.activeTab:
The activeTab permission is used only when the user opens the extension popup for the current YouTube tab. It lets the popup show page-specific counters and status for the active page.

permission.storage:
The storage permission saves local extension settings and counters, such as whether the badge counter and detailed breakdown are enabled. This data stays in the browser and is not transmitted.

host_permission:
The extension needs access to https://www.youtube.com/* because its single purpose is to work on YouTube pages. It detects YouTube Mix recommendations, removes those Mix cards, and cleans Mix playlist parameters from YouTube watch URLs. It does not access non-YouTube websites.
