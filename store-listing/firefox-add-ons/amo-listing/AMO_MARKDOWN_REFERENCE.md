# AMO Markdown Reference

Some fields in an add-on listing support a limited set of Markdown.

| Style | Syntax | Example |
| --- | --- | --- |
| Bold | `** **` or `__ __` | `**This is bold**` |
| Italic | `* *` or `_ _` | `_This is italicized_` |
| Links | `[Text](Link)` | `[Click me!](https://addons.mozilla.org/)` |
| Abbreviations | `*[abbr]: Abbreviation` | `HTML is fun!` followed by `*[HTML]: Hyper Text Markup Language` |
| Blockquote | `>` | `> This is a blockquote.` |
| Code | Triple backticks | ```` ```js
console.log("Hello, world!");
``` ```` |
| Unordered lists | `-`, `*`, or `+` | `- List item` |
| Ordered lists | `1.`, `2.`, `3.`, ... | `1. List item` |
