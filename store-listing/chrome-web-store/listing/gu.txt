YouTube મિક્સ બ્લૉકર YouTube પૃષ્ઠોમાંથી YouTube મિક્સ ભલામણોને દૂર કરે છે અને મિક્સ વૉચ URL ને સાફ કરે છે જેથી વિડિઓ નિયમિત વિડિઓ લિંક્સ તરીકે ખુલે.

તે શાંત અને ન્યૂનતમ રહેવા માટે ડિઝાઇન કરવામાં આવ્યું છે: જ્યાં સુધી તમે પોપઅપ ખોલવાનું પસંદ ન કરો ત્યાં સુધી કોઈ ફીડ રિપ્લેસમેન્ટ નહીં, તેની પોતાની કોઈ ભલામણો નહીં, અને કોઈ ધ્યાન ખેંચે તેવું UI નહીં.

લક્ષણો

- ભલામણોમાં YouTube મિક્સ કાર્ડ્સને અવરોધિત કરે છે.
- શોધ પરિણામોમાં YouTube મિક્સ કાર્ડ્સને અવરોધિત કરે છે.
- વોચ-પેજ સાઇડબાર ભલામણોમાં YouTube મિક્સ કાર્ડ્સને અવરોધિત કરે છે.
- મિક્સ પ્લેલિસ્ટ પેરામીટર્સ દૂર કરીને મિક્સ વોચ URL ને સાફ કરે છે.
- અવરોધિત મિક્સ અને સાફ કરેલ URL માટે વૈકલ્પિક કાઉન્ટર્સનો સમાવેશ થાય છે.
- બેજ કાઉન્ટરને ડિફોલ્ટ રૂપે અક્ષમ રાખે છે.

ગોપનીયતા

આ એક્સ્ટેંશન ફક્ત YouTube પૃષ્ઠો પર ચાલે છે. તે તમારો બ્રાઉઝિંગ ડેટા એકત્રિત, વેચાણ અથવા ટ્રાન્સમિટ કરતું નથી.
