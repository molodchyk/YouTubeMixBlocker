# Chrome Web Store Review Responses

Paste-ready developer replies for reviews after the 1.5 release.

## General Short Reply

Thanks for the report. YouTube changed its Mix markup, and older versions stopped catching some placements. Version 1.5 updates Mix detection for recommendations, search results, and watch-page sidebars, and also cleans Mix watch URLs. Please update the extension, reload YouTube, and let me know on GitHub if you still see Mix cards.
