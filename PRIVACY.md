# Privacy Policy

YouTube Mix Blocker does not collect, transmit, sell, or share personal data or browsing data while it is installed and running.

The extension runs only on YouTube pages. It detects and removes YouTube Mix recommendations and cleans YouTube Mix watch URLs locally in the user's browser.

## Permissions

- `activeTab` - used only when the user opens the extension popup for the current YouTube tab, so the popup can show page-specific counters and status.
- `storage` - used to save local extension settings and counters, such as whether the badge counter is enabled and how many Mix cards or Mix URLs were handled.
- `https://www.youtube.com/*` - used because the extension's single purpose is to detect and remove YouTube Mix recommendations and clean Mix watch URLs on YouTube pages.

## Stored Data

Settings and counters are stored in local browser extension storage. This data stays on the user's device and is used only to provide the extension's features.

## Network Behavior

YouTube Mix Blocker does not make network requests while it is installed and running. It does not use analytics, ads, tracking, remote code, external servers, or content scripts on non-YouTube sites.

## Optional Uninstall Feedback

After uninstalling the Chrome version, Chrome may open an optional feedback page at `https://molodchyk.com/youtube-mix-blocker/uninstall/`. The extension sets only generic URL parameters for `source=chrome`, the extension version, and the Chrome UI language. It does not send user identifiers, browsing data, settings, counters, YouTube page URLs, or YouTube content.

Submitting the feedback form is voluntary. The form is intended to be processed by Formspree and should not include personal data unless the user chooses to provide it.
